Política de Privacidade

Última atualização: 22 de Outubro de 2025

Esta Política de Privacidade descreve como "Policia Civil - Ações" ("nós", "nosso" ou "o Bot") coleta, usa e armazena informações quando você usa nosso Bot no Discord.

1. Coleta de Dados

Para operar e fornecer suas funcionalidades, o Bot pode coletar e armazenar as seguintes informações:

  • IDs do Discord: Coletamos seu ID de usuário do Discord e o ID do servidor (guild) onde o Bot é usado.
  • Conteúdo de Comandos: Podemos armazenar o conteúdo dos comandos que você executa, especificamente aqueles relacionados à "marcação de ações", para garantir o funcionamento correto do Bot e manter um registro das operações.
  • Logs de Uso: Podemos registrar informações sobre como o Bot é usado (comandos executados, erros) para fins de depuração, melhoria e segurança.

Não coletamos intencionalmente nenhuma informação de identificação pessoal (PII) além dos IDs fornecidos pelo Discord, nem coletamos o conteúdo de mensagens que não sejam comandos diretos para o Bot.

2. Uso dos Dados

Usamos os dados coletados para:

  • Fornecer, manter e melhorar a funcionalidade do Bot.
  • Processar as "ações" e comandos solicitados por você.
  • Monitorar o uso, identificar problemas e prevenir abusos.
  • Responder a solicitações de suporte.

3. Compartilhamento e Armazenamento de Dados

Nós não vendemos, alugamos ou compartilhamos seus dados pessoais com terceiros para fins de marketing.

Os dados coletados são armazenados em servidores seguros. Reteremos os dados apenas pelo tempo necessário para cumprir os propósitos descritos nesta política ou conforme exigido por lei.

4. Seus Direitos

Você tem o direito de solicitar a exclusão dos dados associados ao seu ID de usuário ou servidor. Para fazer isso, entre em contato conosco.

5. Contato

Se você tiver alguma dúvida sobre esta Política de Privacidade, entre em contato conosco através do Discord (Usuário: xablau_sincero) ou pelo e-mail: contato.bot.pcacoes@gmail.com.
